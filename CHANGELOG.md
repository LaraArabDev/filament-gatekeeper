## [1.0.0] - 2025-01-01

### Added
- Initial release of FilamentGatekeeper
- Role and permission management UI for Laravel Filament 3.x
- Spatie Laravel Permission integration
- Multi-guard support
- HMVC modules compatibility
- Performance caching layer
- Filament panel plugin registration via `GatekeeperPlugin`
